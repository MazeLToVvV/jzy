取整

golang中的向上取整，向下取整，以及四舍五入

概述

  • 官方的math包只给了向上和向下取整，分别是math.Ceil()[向上取整]，math.Floor()[向下取整]，并没有四舍五入的直接方法，但是我们可以通过向下取整变相的实现四舍五入。

用法

package main
 
import (
	"fmt"
	"math"
)

func main(){
    x := 1.1
    
    fmt.Println(math.Ceil(x))  // 输出为2
    fmt.Println(math.Floor(x))  // 输出为1
}

四舍五入

  • Word’s cheap, show u my code
package main
 
import (
	"fmt"
	"math"
)

func main(){
    x := 1.1
    y := 1.6

    fmt.Println(math.Floor(x + 0.5)) // 输出为1
    fmt.Println(math.Floor(y + 0.5)) // 输出为2    
}
